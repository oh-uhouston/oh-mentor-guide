### 10:00 am to 10:30 / Laptop Setup and Introductions

> Meet your mentors and fellow students and get your laptop ready for the day.

 - User interface design
 - Code review
 - Logos and branding
 - Translation
 - Quality assurance
 - Sysadmining
   - Website management
 - Legal advice
 - User support
 - Publicity
 - Testing
 - Community management
 - Creating tutorials
 - ... and, yes, coding.
